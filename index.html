Versão Beta 1.0 - Disponível para Uso

Instruções para uso:

Extraia os arquivos para qualquer pasta desejada em seu computador

Abra o excel e coloque os perfis que você deseja, todos sempre na coluna A e sempre embaixo do outro, não pode ter linhas em branco entre um intervalo de uma e outra

Salve o excel, feche-o

Abra o RPA - Hunting.exe

Escolha o nome da planilha onde serão salvas as informações

Deixe o programa rodar até ele finalizar todas as consultas, no final, ele vai gerar um excel com as informações dos perfis que você desejou.

Essa versão está preparada para:

-> Caso o usuário esqueça de colocar a barra na listagem, o programa completa

-> Caso o usuário coloque um link inexistente (relacionado ao linkedin) ele pula e passa para o próximo

-> Caso o ponto flutuante esteja na 3, vai dar certo.

O que precisa ser feito:

-> Melhorar a precisão da promoção das pessoas.
